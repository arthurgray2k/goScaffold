# Security Policy

## Supported Versions
We currently support the latest major release of `goScaffold`. If you are using an older version, please upgrade to the latest release to ensure you have the latest security patches.

| Version | Supported          |
| ------- | ------------------ |
| > 0.x.x | :white_check_mark: |
| < 0.x.x | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within `goScaffold`, please do NOT open an issue. Instead, privately report the vulnerability to the project maintainers via the GitHub Security Advisory feature (if enabled) or directly reach out to the project owner.

We will try to review all security reports and issue a patch as soon as possible.
